# Doge画质助手
专门给和平精英修改画质的应用，学习Flutter随便开发的，仅支持安卓版本，没什么技术含量 :)

## 应用功能
* 一些简单的画质修改
* 完成特定任务即可使用
* 还有花里胡哨没用的功能
* 支持安卓13设备画质修改
